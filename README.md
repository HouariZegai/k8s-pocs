# Kubernetes

## Prerequisites
Before getting started, make sure you to follow these steps in order to install **Minikube** with **kubectl**: [Minikube Start](https://minikube.sigs.k8s.io/docs/start/)

## Some Commands

```shell
# Start Minikube:
minikube start
# Check status
minikube status

```
